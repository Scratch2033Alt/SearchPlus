W3C API

現在、仕様、グループ、組織、およびユーザーに関するデータを公開しています。当初はJSON形式で提供していますが、別の形式も準備中であり、時間をかけて公開する情報を拡大していく予定です。

Githubでより詳しく

  • ヘルプを見る

    スタックオーバーフロー  にはW3C 標準と W3C 開発者ツールに焦点を当てた、活発で素晴らしいコミュニティがあります。
    W3C コミュニティに質問するには、w3c タグを使用してください。

    質問する

  • 貢献

    開発者は、既存のツールや提案された将来のプロジェクトにコードを貢献することで参加することができます。

    テスト、バグ報告、提案、あるいは良いドキュメントを作る手助けも重要です！

    各プロジェクトを始めるための詳しい情報は、それぞれのウェブページにあります。

    Githubでプロジェクトを探す

  • 寄付

    優れたツールを開発し、サポートし、ホストし、維持するには多くのコストがかかります。

    あなたの協力があれば、私たちはツールやバリデーターをより良く、より速く、より親しみやすくし、ウェブ全体のための素晴らしい無料サービスを維持することができます。
