はじめに

World Wide Web Consortium (W3C) は、国際的な公益非営利団体であり、メンバー組織、専任スタッフ、および一般市民が協力して、ウェブの長期的な成長を保証し、人類の利益のために働くウェブ標準を開発しています。

World Wide Web Consortium, Inc. は、World Wide Web Consortium の法的本拠地を提供するために米国で法人化された 501(c)(3) です。法的な本拠地として、この法人は独自の細則と理事会を持っています。

使用上の注意： World Wide Web Consortium, Inc." という完全な法人形態は、法人組織に言及する場合にのみ使用されます。私たちを参照する方法については、こちらをお読みください。

理事会

W3C 理事会 -2022年9月に初開催- は、World Wide Web Consortium の公益非営利組織の統治機関です。理事は、W3C の戦略的方向性に関する最終的な権限を有し、ウェブの可能性を最大限に導くという W3C の使命を確実に実行する法的義務を負い、W3C 全体に対する受託者責任を負います。

理事会の公開ページをご覧ください
