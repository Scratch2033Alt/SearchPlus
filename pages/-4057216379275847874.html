在Web上实现更简单、更安全的支付和电子商务。

Web正变革支付方式

为确保Web不断发展以满足新的行业需求，我们正在召集电子商务公司、支付服务提供商、银行卡网络、银行、浏览器供应商及其他相关方，共同塑造Web支付和电子商务的发展方式。

联系我们以加入W3C，开始贡献您的力量 了解W3C在Web支付领域的最新工作（2023年4月）

我们的社区专注于

  • 降低购物车弃置率；
  • 简化身份验证并减少欺诈；
  • 促进数字支付方法的创新；
  • 探讨新兴法规对Web架构和功能的影响；
  • 与FIDO和EMVCo等其他标准化工作的协调一致；
  • 专注于交付API以提高Web支付的便利性和安全性；
  • 提高Web支付领域的整体安全性；
  • 汇集各方意见以指导下一阶段的Web支付发展。

活跃组

这些小组致力于创建和实现软件及API，以使Web支付更加简便和安全，或者提供一个讨论和实施新标准的平台，同时也允许成员发现并解决W3C标准和API在互操作性方面的差距。

Web AuthenticationWorking group

The mission of the Web Authentication Working Group is to define a client-side API providing strong authentication functionality to Web Applications. 访问Web Authentication Working Group

Web PaymentsWorking group

The mission of the Web Payments Working Group is to make payments easier and more secure on the Web. 访问Web Payments Working Group

Web Payment SecurityInterest group

The mission of the Web Payment Security Interest Group is to enhance the security and interoperability of various Web payments technologies. The group pursues its mission by creating a forum for organizations to define areas of collaboration and identify gaps between existing technical specifications in order to increase compatibility among different technologies. 访问Web Payment Security Interest Group

Anti-FraudCommunity group

The mission of the Anti Fraud Community Group is to identify and define scenarios involving fraud and unwanted traffic, as well as to incubate and develop web features and APIs to address those scenarios while improving user security, privacy, and accessibility. Fraud and unwanted traffic can include web activity perpetrated by botnets, attackers impersonating users, and other activity that intends to harm users or compromise web services. The group welcomes participation from anti-fraud service providers, common targets of unwanted traffic, browser vendors, web privacy advocates, web application developers, web hosting and cloud service providers, and other interested parties. The Community Group will discuss issues that server operators and anti-fraud service providers face in this area and ideas for new web features and APIs intended to be implemented in browsers or similar user agents. 访问Anti-Fraud Community Group

为什么加入？

自1994年以来，我们通过开发确保Web长期发展的协议和指南，引领万维网尽展其全部潜力。

加入这些小组，您可以讨论和推动W3C制定的标准。这些小组的讨论涉及当前和未来的标准，并不断致力于改善Web环境。您将有机会参与引领Web的未来发展。

我们的价值观植根于为所有人构建Web

Web的社会价值在于提供人类沟通、商业活动以及知识共享的机会。W3C的主要目标之一是让所有人都能享受这些益处，不论其硬件、软件、网络基础设施、母语、文化、地理位置或身体和心理能力如何。
